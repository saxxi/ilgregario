# 0016 — Users/Teams List Page (`/users/`)

## Goal
Create `GET /users/` — a teams directory showing every manager with their full athlete roster, rank, and points.

## Screens to build
- `templates/dashboard/users.html` — card grid, one card per manager

## Data shape needed (`get_all_users_with_rosters`)
Each user entry:
```python
{
  "username": str,
  "initials": str,
  "color": str,
  "rank": int,
  "total_points": int,
  "gap": int,
  "streak": str,
  "athletes": [
    {"full_name": str, "flag": str, "team": str, "total_points": int}
  ]
}
```

## Implementation steps
1. Add `get_all_users_with_rosters()` to `db_queries.py`
2. Add `GET /users/` route to `routers/dashboard.py`
3. Create `templates/dashboard/users.html`
